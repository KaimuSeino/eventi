##
Eventiは、AKATSUKIプロジェクトの一環である「秋田若手人材発掘・育成プロジェクト」の第1期生のBチームで作成し、最終成果報告会で最優秀賞を受賞したプロダクトです。


## 使用技術

<ul>
    <li>フレームワーク：React 18 / NextJS</li>
    <li>データベース：Supabase(postgresql)</li>
    <li>ORM：Prisma</li>
    <li>認証認可：clerk</li>
    <li>UI：shadcn ui</li>
    <li>画像保存：uploadthings</li>
    <li>メール：Resend</li>
</ul>
